# 重建框架

重建框架从对侧半球估计每个解剖半球。它不是将侧化视为简单的左右相减，而是学习半球之间的非线性映射，然后研究该映射无法解释的残余部分。

## 概念步骤

1. 将预处理灰质图分割为左右半球。
2. 训练方向特定的重建模型：
   - 左到右重建
   - 右到左重建
3. 将训练好的模型应用于保留受试者。
4. 将每个实际目标半球与其重建对应物配对。
5. 计算残差图和下游汇总。

## 模型系列

HemiSpec 稿件遵循原始 Patterns 框架中使用的 DGN/上下文编码器风格重建策略。公开文档应将模型描述为跨半球 DGN，除非特定实现页面正在讨论架构细节。

## HemiSpec 新增内容

HemiSpec 将重建输出打包为可复现的工作流：图计算、ROI 汇总、验证、报告和下游表型分析。
