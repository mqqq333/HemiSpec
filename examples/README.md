# Examples

Examples in this folder must be public-safe.

- `synthetic_quickstart/` generates toy NIfTI maps and validates the CLI contract without private MRI data.
- `input_sample/` is reserved for local or explicitly approved examples; real subject-level NIfTI files are ignored by git.
- `run_example.sh` is a legacy shell example and should be kept aligned with the `hemispec` CLI.
